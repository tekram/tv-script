## Key Features Overview

### Swing Dashboard (Main)
- Market cap & float calculations
- ADR% and ATR measurements
- Volume analysis with relative volume
- EMA clouds (5-12, 34-50)
- VIX clouds for sentiment
- Market breadth indicators

### Bottom Right Table (Component)
- Focused EMA cloud analysis
- 10-minute timeframe optimization
- Color-coded bullish/bearish signals
- VIX contrarian indicators

## Technical Requirements

- TradingView account with Pine Script v6
- Minimum plan supporting 24+ request calls (main dashboard)
- Any plan for EMA component (2 calls only)
